# 扫描大图片技能

## 技能描述
扫描当前项目目录，查找所有超过指定尺寸的图片文件，并生成详细的报告。

## 执行步骤

1. **询问用户扫描参数**
   - 使用 AskUserQuestion 询问用户要扫描的图片尺寸阈值（宽度或高度）
   - 提供常用选项：1920x1080、2560x1440、3840x2160、自定义

2. **扫描项目中的图片文件**
   - 使用 Glob 工具查找常见的图片格式：**/*.{png,jpg,jpeg,gif,bmp,webp,svg,ico,tiff}
   - 如果在 git 仓库中，使用 Bash 工具执行 `git ls-files` 获取被追踪的文件，然后过滤图片文件

3. **检测图片尺寸**
   - 对于每个图片文件，使用 Bash 工具调用系统命令获取尺寸信息：
     - macOS: `sips -g pixelWidth -g pixelHeight <file>`
     - Linux: `identify -format "%wx%h" <file>` (需要 ImageMagick)
     - 或使用 Python 脚本读取图片尺寸

4. **过滤大图片**
   - 根据用户指定的阈值，筛选出超过该尺寸的图片
   - 记录每个图片的：文件路径、宽度、高度、文件大小

5. **生成报告**
   - 统计总图片数量、大图片数量、大图片占比
   - 按尺寸降序排列大图片列表
   - 以表格形式展示结果
   - 提供优化建议（如压缩、裁剪、使用 WebP 格式等）

6. **执行通知命令**
   - 任务完成后执行：`play -n synth 0.5 sine 1000-100 vol 0.2`

## 注意事项
- 跳过 node_modules、.git、dist、build 等常见忽略目录
- 处理可能出现的错误（如损坏的图片文件）
- 支持相对路径和绝对路径
- 对于 SVG 文件，解析 viewBox 或 width/height 属性
